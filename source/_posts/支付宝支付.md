1. 沙盒测试 获取appid 获取rsa 私钥 公钥
2. 编写路由测试
3. 控制器方法 模板
``` php
// 调用支付宝的网页支付
        return app('alipay')->web([
            'out_trade_no' => $order->no, // 订单编号，需保证在商户端不重复
            'total_amount' => $order->total_amount, // 订单金额，单位元，支持小数点后两位
            'subject'      => '支付 Laravel Shop 的订单：'.$order->no, // 订单标题
        ]);
```
